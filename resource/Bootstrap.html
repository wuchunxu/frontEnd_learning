  • 编写响应式网站

    响应式

    如何编写响应式网站

    声明viewport(视口)元标签

    <meta name="viewport" content="width=device-width,initial-scale=1">

    早期3G手机为浏览器大尺寸网页，只能强行页面缩小,导致图片/文字/链接很小。IOS提出“视口”概念，视口用来盛放网页内容，尺寸可以指定，超过部分可以滑动网页来查看。

    width=device-width宽度等于设备宽度

    initial-scale=1初始化缩放比例为1

    使用流式布局

    float:left;/display:inline-block;

    使用相对尺寸

    容器、文字、图片都使用相对尺寸

    .container{
    width:50%;
}
body{
    font-size:2em;
}
img{
    width:25%;
}

    【重点】媒体查询技术(CSS3--Media Query)

    Media:指浏览网页的设备，如screen(pc/pad/phone),print,tv。

    Media Query:查询当前浏览网页设备类型、特性(解析度、对比、尺寸、手持方向等),而选择性执行某些css代码，而忽略另外一些。

    使用媒体查询有两种方法:

    (1)根据媒体查询的结果执行不同的CSS文件；
    <link  media="screen and (min-width:768px) and (max-width:991px)" rel="stylesheet" href="1.css"/>
<link  media="print" href="2.css"/>
    (2)根据媒体查询结果执行CSS片段汇总不同的部分。
    /*pad*/
@media screen and (min-width:768px) and (max-width:991px){
    //css code
}
    PC >=992px
    pad 768~991px
    phone <=767px

  • 全局CSS样式

    全局样式

    按钮.btn

    默认颜色 .btn-default
    五种颜色 .btn-info/success/danger...
    大小 .btn-lg/md/sm/xs
    块级按钮 .btn-block

    图片

    圆角图片 .img-round
    圆形图片 .img-circle
    缩略图片 .img-thumbnail
    响应式图片 .img-response

    文本

    文本色 text-success/danger...
    背景色 .bg-danger/warning/success
    对齐 .text-left/center/right/justify
    转换大小写 .text-uppercase
    lowercase/capitalize

    注意：bootstrap中大小写转换用的是CSS3新属性：text-transform:lowercase

    列表

    无提示符号 .list-unstyle
    行列列表 .list-inline

    Bootstrap中屏幕分类

    大型PC lg >=1200
    中型PC md 992~1199
    pad sm 768~991
    Phone xs ~767

    表格.table

    带边框的表格 .table-bordered
    隔行变色 .table-striped
    带悬停变色 .table-hover
    响应式表 .table-responsive

    注意：.table-responsive应该添加在table的父元素上，如果添加在表格本身上，不会报错。

    不过，可以用Bootlint来检测。Bootlint是Bootstrap官方提供HTML检测工具，检测出网页中常见的HTML/CSS错误，还能检测class层级结构错误!
