  • 定位

    定位

    定位的属性和取值

    属性：position:static/relative/absolute/fixed

    属性取值：top/right/bottom/left

    static 默认值
    relative 相对定位
    absolute 绝对定位

    1) 绝对定位会脱离文档流，不占空间；

    2) 相对于离它最近的定位祖先元素

    3) 定位元素可能出现堆叠效果，用z-index调整层级

    使用场景：如弹出菜单
    fixed 固定定位

    固定定位相对于body，位置不随滚动条发生变化
    注意，只有已定位的元素才能使用z-index;父子元素间，z-index无效，子元素永远在父元素之上

  • 生成内容

    生成内容

    CSS动态生成内容

    使用CSS动态地向某元素中插入一段内容,:before {}, :after {}

    可以插入文本，用content:""，也可以插入图片content:url()

    //使用伪元素
p:before {
	content:"";
}
p:after {
	content:"";
}

    应用

    清除浮动

    在父元素最后添加一个空的块元素，并清除浮动，注意.clear挂在父元素上

    .clear:after {
	content:"";
	display:block;
	clear:both;
}

    子元素外边距溢出

    在子元素之前添加空的table子元素

    //CSS
* { margin:0;padding:0; }
div { margin-top:30px;height:200px;width: 300px;background-color:#eee; }
p { margin-top:20px; width:100px; height:100px; background-color:gray; }
div:before{ content:""; display: table; }
//html代码
<div>
	<p></p>
</div>
