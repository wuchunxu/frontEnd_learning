  • 弹性布局--flex-layout

    弹性布局--flex-layout

    弹性布局的相关概念

    名词释义
    弹性布局的容器 简称“容器”
    弹性布局的项目 简称“项目”，要实现布局效果的元素
    主轴

    项目排列方向所在的线轴称为主轴

    若项目按X轴排列，则X轴为主轴；反之，若项目按Y轴排列，则Y轴为主轴
    交叉轴

    与主轴交叉的轴为交叉轴

    使用弹性布局

    /*将容器设置为弹性即可*/
.flex-container{
	display:flex;
	flex-direction:row;/*水平排列，左边开始*/
	flex-wrap:no-wrap;/*不换行*/
}

    注意点:将元素变为flex容器后，1) 其所有子元素都变为项目; 2) 子元素的float、clear、vertical-align将失效; 3) 容器的text-align会失效

    容器的相关属性

    flex-direction flex-direction规定容器的主轴及排列方向
    • row:横向排列，起点在左边
    • row-reverse:横向排列，起点在右侧
    • column：纵向排列，起点在上
    • column：纵向排列，起点在下方
    flex-wrap 规定，当一个主轴排列不下所有项目时，是否需要换行。取值如下：
    • no-wrap:默认值，空间不够也不换行，项目会自动缩小
    • wrap：换行
    • wrap-reverse：换行并反转
    flex-flow

    是flex-direction和flex-wrap的简写

    flex-flow:row wrap;
    justify-content 定义项目在主轴方向上的对齐方式，取值如下：
    • flex-start在主轴起点对齐
    • flex-end在主轴终点对齐
    • center:主轴上居中对齐
    • space-between:两端对齐，常用
    • space-around:元素两端边距相等对齐，不常用
    align-items 定义项目在交叉轴方向上的对齐方式，取值如下：
    • flex-start在交叉轴起点对齐
    • flex-end在交叉轴终点对齐
    • center:交叉轴居中对齐
    • baseline:交叉轴上基线对齐
    • stretch:拉伸，也即，若项目未设置尺寸，在交叉轴上占满所有空间

    项目的相关属性

    order 定义项目的排列顺序，值越小，离起点越近，默认值为0
    flex-grow

    定义项目的放大比例，如果容器有足够的剩余空间，可以指定项目如何放大

    取值整数数字，默认值为0，不放大；取值越大，占据更多的剩余空间
    flex-shrink

    定义项目的缩小比例【空间不足时，如何缩小】

    取值：默认值为1，空间不足时，等比缩小；取值为0，则不缩小
    align-self 定义当前项目在交叉轴上的对齐方式
    • flex-start
    • flex-end
    • center
    • baseline
    • stretch

  • 转换--transform

    转换--transform

    转换的相关属性

    函数功能
    位移
    translate    		 translate(x) 元素在X轴上位移x
    translate(x,y) 元素在X轴上位移x，Y轴上位移y
    translateX(x) 元素在X轴上位移x
    translateY(y) 元素在Y轴上位移y
    缩放
    scale    		 scale(value)

    默认值为1，大于1时，放大；小于1时，缩小；

    负值：放大，但水平和垂直方向都翻转180°
    scale(x-v,y-v) 分别定义X和Y方向的缩放比例
    旋转
    rotate    		 rotate(xdeg)

    transform: rotate(45deg)

    注意：转换原点会影响转换效果
    倾斜 skewX(xdeg) 将元素沿X轴方向产生倾斜效果,x>0时，Y轴向左倾斜xdeg
    skewY(ydeg) 将元素向Y轴产生倾斜效果
    skew(x,y)

    转换原点

    取值含义
    值的形式数值、百分比、关键词
    值的个数

    2个值表示X轴和Y轴上的位置

    3个值表示XYZ轴上的位置

    3D转换

    透视距离
    perspective

    模拟人的眼睛到3D转换元素之间的距离（Z轴）

    要实现3D效果，需要在父元素上设置perspective属性
    3D旋转

    属性：transform

    • rotateX(xdeg):以X轴为中心轴，旋转
    • rotateY(ydeg):以Y轴为中心轴，旋转
    • rotateZ(zdeg):以Z轴为中心轴，旋转
    • rotate3D(x,y,z,ndeg):3D旋转n°,
      x,y,z大于10时，表示参与旋转，取值为0，不参与旋转，rotate3d(1,0,0,45deg)

  • 过渡--transition

    过渡--transition

    什么是过渡

    过渡是指，CSS的属性值在给定时间内平缓变化的效果。如果没有过渡，属性值会瞬间改变。

    过渡的写法

    transition: all 1s ease 0s;
//transition: property duration timing-function delay;

    transition的相关属性

    属性含义
    transition-property all 只要能使用过渡属性的，都使用过渡属性

    transition-property:all;
    具体 允许使用过渡的属性：颜色、取值为数字的属性、转换、阴影、渐变、visibility
    • transition-property:background;
    • transition-property:border-radius;
    transition-duration完成过渡所需要的时间
    transition-timing-function
    • ease：默认值，速度由慢到快，再到慢；
    • linear：匀速
    • ease-in:慢速开始，加速结束
    • ease-out:快速开始，慢速结束
    • ease-in-out:慢速开始和结束，中间先加速后减速
    transition-delay 延迟时间，0s/1s...

  • 动画--animation

    动画--animation

    什么是动画

    将过渡效果连起来就是动画，通过“关键帧”控制动画步骤的执行。

    关键帧keyframe表示动画执行的时间点以及在该时间点的样式

    如何使用动画

    声明动画-->调用动画

    声明动画

    @keyframes my_animate {
	0%{}
	50%{}
	100%{}
}

    兼容性 

    /*兼容*/
@-webkit-keyframes my_animate {}
@-ms-keyframes my_animate {}
@-o-keyframes my_animate {}
@-moz-keyframes my_animate {}

    调用动画

    animation: name duration timing-function delay iteration-count direction

    动画的相关属性

    属性作用
    animation-name
    animation-duration
    animation-timing-functionease/linear/ease-in/ease-out/ease-in-out
    animation-delay5s/5000ms
    animation-iteration-count1/2...infinite
    animation-directionnormal/reverse/alternate(奇数次正向、偶数次反向)
    animation-fill-mode 规定动画在播放前或播放后的状态
    • none:默认值
    • forwards：动画播放完，将保持在最后一个帧的状态上
    • backwards：动画播放前，延迟时间内，动画保持在第一个帧的状态上
    • both：forwards和backwards的结合体
    animation-pause-state指定动画是处于播放还是暂停状态，取值paused、running
