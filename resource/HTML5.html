  • Canvas绘图

    Canvas绘图

    了解绘图

    网页中的特效，实时走势图，统计图，网页游戏，地图应用使用绘图技术，有三种绘图技术:

    svg 绘图

    2D矢量绘图技术，2000年出现，后纳入H5
    canvas绘图

    2D位图绘图技术，H5提出的绘图技术
    webgl绘图

    3D绘图技术，尚未纳入H5标准

    canvas

    canvas:画布，是h5提供的2D绘图技术。canvas标签在浏览器中默认300*150 inline-block，画布宽度和高度只能使用html/js来赋值，不能用css样式赋值。

    每个画布上有且只有一个画笔对象

    var ctx = canvas.getContext("2d");//获得画笔

    绘制矩形

    ctx.lineWidth = 1;         //描边宽度 
ctx.fillStyle = "#000";      //填充样式
ctx.strokeStyle = "#000";   //描边样式
ctx.fillRect(x,y,w,h)        //填充一个矩形
ctx.strokeRect(x,y,w,h);    //描边一个矩形
ctx.clearRect(x,y,w,h);     //清除一个矩形范围内所有绘图

    绘制文本

    /*********文本绘制*************/
ctx.baseLine = "top";
ctx.font = "12px sans-serif";//文本字体大小
ctx.fillText(str,x,y);      //填充文本
ctx.strokeText(str,x,y)     //描边一段文本

    绘制渐变色

    var g = ctx.createLinearGradient(x1,y1,x2,y2);
g.addColorStop(0,"red");
g.addColorStop(0.5,"green");
ctx.fillStyle = g;
    练习：渐变对象柱子：宽50，高350
    练习：sales.php创建一组数值

    使用canvas进行绘图--路径

    由多个坐标点组成的任意形状，路径不可见，可用于"描边","填充","裁剪"

    ctx.beginPath(); 开始一条新路径
    ctx.closePath(); 闭合当前路径
    ctx.moveTo(x,y) 移动到指定点
    ctx.lineTo(x,y) 从当前点到指定点画直线
    ctx.arc(cx,cy,r,start,end); 绘制圆拱路径
    ctx.stroke(); 对当前路径描边
    ctx.fill(); 对当前路径填充
    ctx.clip(); 使用当前路径裁剪

    canvas--绘制图像

    canvas属于客户端技术，图片保存于服务器上，所以浏览器必须先下载图片再进制绘制，且等待图片下载完成再进行。

    var p3 = new Image();        #创建图片对象
p3.src = "x.jpg";             #下载图片
p3.onload = function(){       #图片下载完成后触发事件
    ctx.drawImage(p3,x,y);     #绘制原始原始大小图片
    ctx.drawImage(p3,x,y,w,h);  #拉伸图片
}

    canvas--变形操作

    canvas绘图中变形技术:可以针对某一个图像/图形的绘制过程进变形:rotate,scale,translate

    ctx.rotate(弧度); 旋转(画笔),轴点画布原点
    ctx.translate(x,y); 将画布的原点平移到指定点
    ctx.save(); 保存画笔状态(原点,角度,颜色...)
    ctx.restore(); 恢复画笔状态到最近一次保存

    SVG绘图

    canvas与svg的区别

    Canvas SVG
    类型 2d位图 2d矢量图
    如何绘图 js代码绘图 标签绘图
    事件绑定 每个图形不是一个元素，无法直接绑定事件 每个图形都是元素，可以直接绑定事件监听
    应用场景 统计图/特效/游戏 图标/地图

    SVG绘图的注意点

    (1)所有图形默认只有填充色（黑色），没有描边色

    (2)SVG图形的样式，可以由元素属性声明，也可以用css形式声明，但只能用SVG专用样式声明。如边框stroke，而不是border

    (3)图形可以使用js来对属性赋值，但不能用HTML DOM形式，只能用核心DOM。r2.x=10;//无效

    (4)动态添加svg图片的方式有，svg.innerHTML="";和创建元素的方式

    var node = document.createElementNS("http://www.w3.org/2000/svg","元素名");

  • Web Worker

    Web Worker

    程序、进程和线程

    程序(program):被CPU执行的代码，存在硬盘中。

    进程(process/Task)：硬盘中的代码加载到内存中，分配空间

    Chrome浏览器线程模型

    一个Chrome进程内，有多个线程“并发”向web服务器发起http请求，以获得所需要的资源。(资源请求线程)

    还有一个线程(UI主线程)，负责绘制网页中内容并且解析js程序。（只有一个）

    解决方案：创建一个新线程(worker)来执行耗时的js任务。

    <script>
    var worker = new Worker("02.js");
</script>

    要注意，浏览器不允许Worker线程操作任何DOM和BOM对象，原因是，如果多个线程操作DOM，页面可能会混乱。因此，jQuery代码不能放在worker线程里。

    Worker线程可以给UI线程发送数据，反之也行。

    /*
    worker线程---->ui线程                
*/
//worker.js
postMessage(str);
//UI线程.html
var worker = new Worker("x.js");
worker.onmessage = function(e){
    //e.data
}
/*
    ui线程---->worker线程
*/
//UI主线程
var worker = new Worker("x.js");
worker.postMessage(str);
//worker线程
onmessage = function(e){
    //操作e.data
}

    项目中Worker使用场景

    (1)只要js中有DOM和BOM就不能用worker

    (2)适用于执行耗时的js任务，复杂计算，加密和解密..

    比较精确执行一段js代码的小技巧

    var start = new Date().getTime();//1 创建一个变量保存当前时间戳
do{
    var end = new Date().getTime();//2 在do while循环中创建时间戳
}while(end-start < 5000);//3 在判断条件中计算，如果循环中时间戳-开始时间戳小于5000，继续执行

    Web storage

    Web storage

    web storage在浏览器中存储当前用户专用数据，如内容定制，样式定制。

    在客户端存储数据可选方式

    # 技术 特性
    1 Cookie 浏览器兼容性好，但不能超过4KB，操作复杂
    2 Flash 依赖于FLASH播放器
    3 H5 WebStorage 不能超过8M，操作简单
    4 indexDB 可存储大量的数据，还不是标准技术

    webstorage的对象

    window.sessionStorage

    类数组对象，会话级数据存储。在浏览器进程中分配内存，存储本次会话使用的数据，该会话中所有页面共享此数据，当浏览器关闭，该数据消失。

    在同一个会话汇总所有页面共享的数据，如昵称。

    sessionStorage[key] = value;        //保存数据
sessionStorage.setItem(key,value);  //保存数据
var value = sessionStorage[key];
var value = sessionStorage.getItem(key);
sessionStorage.removeItem(key);     //删除一组数据
sessionStorage.clear();             //清除所有数据
sessionStorage.length;              //数据个数
var key = sessionStorage.key(i);    //获取第i个key

    Chrome中查看session storage:application--storage

    利用sessionStorage对象可以做出登录、注销客户端效果。
    //login.html
sessionStorage.setItem("uname","dingding");
href.location = "index.html";
//index.html
var uname = sessionStorage.getItem("uname");
if(uname){
    userMsg.innerHTML = `欢迎回来，${uname},<a href="logout.html">退出</a>`;
}
//logout.html
sessionStorage.removeItem("uname");
location.href = "index.html";

    localStorage

    在硬盘中存储用户的专用数据，可供此次会话中后续页面共同使用，即使浏览器关闭也不会消失。用来存储所有会话共享数据，但很少使用。

  • WebSocket

    WebSocket

    HTTP协议

    HTTP协议的劣势

    超文本传输协议，用来传输网络资源（html、css、image、mp3、mp4）。是“请求-响应”模型，只有客户端发起请求消息，服务器才会返回响应消息；一个请求只能得到一个响应。此模型在有些场景有些力不从心--比如实时股票走势图。

    解决方案

    长轮询/心跳请求

    定时器+ajax;

    WebSocket协议

    该协议属于“广播-收听”模型，客户端连接到服务器就不再断开了，永久连接，双方都可以随时向对方发送消息，WS在实时走势应用有着特别的优势。

    WS服务器的应用

    监听指定端口，接收客户端请求，向对方发送消息，并接收消息，可以使用PHP/JAVA/Node.js等编程实现。
    WS客户端应用

    主动发起连接请求，保持永久连接，向对方发送消息，并接受消息，可以用HTML5对象。

    使用H5创建WS客户端应用

    //连接ws服务器
var socket = new WebSocket("ws://127.0.0.1:9001");
//向服务端发送消息
socket.send(str);
//接收服务端的消息
socket.onmessage = function(e){
    //e.data
}
//断开连接
socket.close();
