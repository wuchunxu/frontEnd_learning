    增强型表单

    新的input type

    h4 text/password/radio/checkbox/submit/button/reset/file/image
    h5 email/url/number/search/range/color/date/month/week

    新的表单元素

    h4 input/textarea/(select/option)/button

    输入建议列表(datalist)

    datalist本身不可见，它为input提供一个建议输入列表，用户可以选取也可以手工输入。

    <datalist id="list3">
    <option>九华山</option>
    <option>便宜坊</option>
    <option>大董</option>
</datalist>

    进度条(progress)

    <progress value="0.1" id="progress" style="width:200px"></progress>

    进度条:

    meter--刻度尺

    用于表示一个值所处的范围，不接接受(红色)，可接受的(黄色)，非常优秀(绿色)

    <meter value="0" min="0" max="100" low="60" high="90" optimum="99" id="score"></meter>

    output--输出

    输出，语义化标签，没有任何外观样式，样式等同于span

    商品单价 商品数量 小计
    ￥3.5 ￥3.5

    <th>商品数量</th>
    <th>小计</th>
    </tr>
<tr>
    <td>￥3.5</td>
    <td><input type="number" value="1"></td>
    <td><output>￥3.5</output></td>
    </tr>
</table>

    视频

    <video src="res/x.mp4"></video>
//兼容性写法
<video>
    <source src="x.mp4">
    <source src="x.ogg">
    <source src="x.webm">
    您的浏览器版本太低，请下载最新版本！
</video>

    video本身是一个300*150的inline-block

    属性 功能
    autoplay 默认false，是否自动播放
    controls 默认false,是否显示播放控件
    loop 默认false,是否循环播放
    muted 默认false,是否静音
    poster:"" 在播放第一帧之间显示海报
    preload

    auto:预加载视频元数据和缓冲一定时长视频

    metadata:仅加载视频元数据(尺寸，时长，第一帧)

    none:不预加载任何数据

    js对象属性

    成员属性 currentTime 当前播放时长
    duration 总时长
    paused 是否处于暂停状态
    volume 当前音量 0~1 0.5
    playbackRate 播放速率 大于1快放,小于1慢放
    成员方法 play() 播放视频
    pause() 暂停暂放
    成员事件 onplay 当视频开始播放时触发事件
    onpause 当视频暂停时触发事件

    音频

    <audio src="res/x.mp4"></audio>

    audio本身是一个300*30的inline-block，单若没有controls属性，则display:none;

  • Canvas

    Canvas绘图

    了解绘图

    网页中的特效，实时走势图，统计图，网页游戏，地图应用使用绘图技术，有三种绘图技术:

    svg 绘图

    2D矢量绘图技术，2000年出现，后纳入H5
    canvas绘图

    2D位图绘图技术，H5提出的绘图技术
    webgl绘图

    3D绘图技术，尚未纳入H5标准

    canvas

    canvas:画布，是h5提供的2D绘图技术。canvas标签在浏览器中默认300*150 inline-block，画布宽度和高度只能使用html/js来赋值，不能用css样式赋值。

    每个画布上有且只有一个画笔对象

    var ctx = canvas.getContext("2d");//获得画笔

    绘制矩形

    ctx.lineWidth = 1;         //描边宽度 
ctx.fillStyle = "#000";      //填充样式
ctx.strokeStyle = "#000";   //描边样式
ctx.fillRect(x,y,w,h)        //填充一个矩形
ctx.strokeRect(x,y,w,h);    //描边一个矩形
ctx.clearRect(x,y,w,h);     //清除一个矩形范围内所有绘图
